# React ToDo List

Projeto de um **ToDo App simples em ReactJS**, desenvolvido para fins didáticos, com foco na **comparação entre as arquiteturas MVC, MVP e MVVM**.

O aplicativo implementa:
- Listagem de tarefas
- Criação de tarefa
- Exclusão de tarefa
